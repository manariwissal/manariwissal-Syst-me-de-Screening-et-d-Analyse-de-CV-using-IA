## Projet : Système de Screening et d’Analyse de CV

Ce projet a été réalisé par Wissal Manari, Loubna Fouzi, et Iham Ettouil, membres de l’équipe « Explorateurs de Savoir ».

L’objectif principal de ce projet est de faciliter le processus de recrutement en automatisant l’analyse et le classement des CV. Le système permet :

Extraction automatique des informations clés : nom, compétences, expériences professionnelles, diplômes.

Nettoyage et prétraitement des données pour uniformiser le contenu des CV.

Classification intelligente des CV selon les catégories métiers à l’aide d’un modèle SVC entraîné sur un corpus de CV.

Interface Web interactive permettant de visualiser les résultats et de télécharger les CV analysés.

Grâce à ce projet, l’équipe a pu démontrer l’application pratique de techniques de machine learning, de traitement du langage naturel (NLP) et de développement Web, afin de créer un outil efficace pour les recruteurs et les entreprises.
